# ⚡ Quick Start Guide - Emergency Alert System

## 🚀 Get Started in 3 Minutes

### Option 1: Use Demo Account (Fastest)

1. **Open the Application**
   - Open `index.html` in your browser
   - Or use the live deployment URL from the Publish tab

2. **Login with Demo Credentials**
   - Email: `demo@emergency.com`
   - Password: `demo1234`

3. **Start Using**
   - You're in! Explore all features immediately

---

### Option 2: Create Your Own Account

1. **Register**
   - Click "Get Started" or "Register"
   - Fill in your details:
     - Full Name
     - Email
     - Phone Number
     - Password (min 6 chars)
   - Click "Create Account"

2. **Add Emergency Contacts**
   - Go to "Contacts" page
   - Click "Add Contact"
   - Fill in contact details
   - Save

3. **You're Ready!**
   - Go to Dashboard
   - Use the big red SOS button when needed

## 🎯 Key Features to Try

### 1. Emergency Alert (Main Feature)
- **Location**: Dashboard page
- **Action**: Click the large red "SOS EMERGENCY" button
- **Result**: Instant alerts to all your contacts

### 2. Add Emergency Contacts
- **Location**: Contacts page
- **Action**: Click "Add Contact" button
- **Tip**: Add at least 3 contacts for best results

### 3. View Alert History
- **Location**: History page
- **Action**: See all past alerts
- **Feature**: Export to CSV

### 4. Customize Your Profile
- **Location**: Profile page
- **Actions**: 
  - Update personal info
  - Change password
  - Toggle dark/light theme
  - Set preferences

### 5. Silent Alert
- **Location**: Dashboard → Quick Actions
- **Action**: Click "Send Silent"
- **Use**: For discreet emergency notifications

## ❓ Common Questions

**Q: Can I cancel an alert after sending?**  
A: Currently, alerts are sent immediately. Future versions will include cancel/update features.

**Q: How many contacts can I add?**  
A: Unlimited! Add as many as you need.

**Q: What if I don't have internet?**  
A: The app requires internet to send alerts. In emergencies without internet, call emergency services directly (100, 108, 101).

**Q: Is my data secure?**  
A: Yes, all data is stored securely. Passwords are hashed.

**Q: Can I use this on multiple devices?**  
A: Yes! Login from any device with your credentials.
